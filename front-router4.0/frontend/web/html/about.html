铂略是中国领先的财务管理及实务技能培训服务供应商，是中国最大的在线财务教育平台。 依托互联网战略，通过铂略财务实务培训平台（www.bolue.cn），利用语义分析系统、智能需求匹配系统、大数据分析推送系统、培训评估分析系统、集团培训管理系统等，努力为企业及个人打造一站式财税培训解决方案。

同时铂略为打造全天候、零差旅、低成本、高品质的财务服务体系，目前已形成在线、线下、移动互联的服务闭环，受数千家企业客户好评，每年有数十万财务专业人士因此受益。

秉承“让专业知识触手可及”的企业使命，以“专注、及时、高效”为服务理念，已累计开发近600门线上及线下课程、撰写近600万字的财务报告、收集及分享数10万条财务资讯、累积并反馈5万多条财税问答。通过体系化的梳理，铂略已形成庞大且专业的企业财税服务数据库。

目前，铂略的课程已受到众多知名国际会计师协会的认可及推荐，诸如特许管理会计师公会（CIMA）、特许公认会计师公会（ACCA）、国际会计师公会（AIA）、澳大利亚公共会计师协会（IPA AU）、美国管理会计师协会（IMA）等已与铂略达成合作。
